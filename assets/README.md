Az oldal három fotót használ. Töltsd fel ide a fájlokat **pontosan ezekkel a nevekkel**
(GitHub: Add file → Upload files), és automatikusan megjelennek. Amíg nincsenek fent,
az oldal egy elegáns helyőrzőt mutat — semmi nem törik el.
