Az `IMG_*` fájlok az eredeti, vágatlan fotók (forrás/tartalék). Ha le szeretnéd
cserélni valamelyik megjelenő képet, írd felül a megfelelő `bettina-*.jpg` fájlt
egy kb. 1100 px széles, álló (4:5) JPG-vel.
